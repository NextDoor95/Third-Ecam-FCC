Web 1.0

In the beginning, Internet users only have one role: to consume the information that is hosted on computer servers. The navigation was purely textual and the queries are very limited.

Web 2.0

Around the year 2000, Web 2.0 began to be talked about, by encouraging interaction between web pages and users. The Internet is understood as a collaborative platform, in which all users participate. Now, in addition to reading, Internet users can also generate information and publish it on websites, user forums, blogs, social networks and wikis (pages that can be edited by any user).

Web 3.0

It refers to the fact that web pages can be related semantically, adding metadata that adds value to the information and improves the search and the possibility of finding information on the web by understanding the meaning of the words.

Web 4.0

The Internet can work predictively, not only receiving orders from the user, but by using artificial intelligence it can anticipate their requests and even predict behavior in the future. Interactions through voice are also generalized, with the use of smart devices that allow spoken searches.
